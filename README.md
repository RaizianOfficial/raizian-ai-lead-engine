# 🚀 AI Lead Generation Agent

Automatically:
- Scrapes leads
- Filters using AI
- Sends daily emails

## ⚡ Features
- Fully automated
- Firebase powered
- AI scoring (Gemini)

## 🛠 Setup
1. Clone repo
2. Add environment variables
3. Run


## 🔐 Required Secrets

* GEMINI_API_KEY
* FIREBASE_PROJECT_ID
* FIREBASE_CLIENT_EMAIL
* FIREBASE_PRIVATE_KEY
* SMTP_HOST
* SMTP_PORT
* SMTP_USER
* SMTP_PASS
* EMAIL_TO
* EMAIL_FROM
* SERP_API_KEY

## 🧠 How it works
Scraper → AI → Filter → Email

## 💰 Use Cases
- Agencies
- Freelancers
- Local business automation
