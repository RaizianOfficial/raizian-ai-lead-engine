# 🚀 AI Lead Generation Agent

Automatically:
- Scrapes leads
- Filters using AI
- Sends daily emails

## ⚡ Features
- Fully automated
- Firebase powered
- AI scoring (Gemini)

## 🛠 Setup
1. Clone repo
2. Add environment variables
3. Run

## 🔐 Required Keys
- GEMINI_API_KEY
- FIREBASE_*
- SMTP_*

## 🧠 How it works
Scraper → AI → Filter → Email

## 💰 Use Cases
- Agencies
- Freelancers
- Local business automation
